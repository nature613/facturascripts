Método para calcular el Margen

El Precio de Venta (PVP) es igual al Coste (CST) mas el Margen Comercial (M), es decir:

      PVP = CST + M

El Margen como ratio se puede calcular de dos maneras:

- Sobre el Precio de Venta

      Mv = ( PVP - CST ) / PVP

      Entonces,

      PVP = CST / ( 1 - Mv )   y   M = CST * Mv / ( 1 - Mv )

      En porcentaje,   mv = 100 * Mv

- Sobre el Precio de Coste

      Mc = ( PVP - CST ) / CST

      Entonces,

      PVP = CST * ( 1 + Mc )   y   M = CST * Mc

      En porcentaje,   mc = 100 * Mc


Se dan las relaciones:

  Mv = Mc / ( 1 + Mc )     y     Mc = Mv / ( 1 - Mv )


Por otro lado, la Rentabilidad (R) de un Artículo es:

  R = M / CST     y en porcentaje,     r = 100 * R
