Últimos cambios de la versión {$fsc->version()}:

  • 27/05/2013:
    • Ahora se puede cambiar la serie de albaranes no facturados.
    • Los asientos de ejercicios ya cerrados se bloquean automáticamente (se marcan como no editable).
    • A los artículos bloqueados se les elimina el stock automáticamente.
    • TPV genérico permite configurar si quieres que se impriman las referencias o las descripciones en los tickets.
    • Las facturas de clientes incluyen la dirección y teléfonos de los mismos.
    • Mejoradas las hojas de estilo.
  • 22/05/2013:
    • Añadido soporte experimental para WooCommerce.
    • Añadido control de duplicados. Ahora cuando se envían dos peticiones de un mismo formulario (de nuevo asiento, nuevo albarán o lo que sea) ya sea por fallo del navegador, por mala leche del usuario o por hacer doble clic sobre el botón guardar, se guarda el primero, pero la segunda petición y siguientes no producen elementos duplicados.
    • Los asientos, albaranes y facturas ahora muestran un mensaje de error si encuentran duplicados.
    • El libro mayor de cada subcuenta ahora incluye las sumas del debe y del haber, para facilitar cálculos.
    • El TPV de restaurante permite ahora imprimir un ticket por cada impresora.
    • Pequeñas modificaciones para dar más información al usuario y ofrecer una experiencia más sencilla.
  • 29/04/2013:
    • Añadido soporte para MySQL.

  • 22/04/2013:
    • Ya se pueden modificar los asientos.
