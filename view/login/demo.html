Potente

FacturaScripts está diseñado para sacarle el máximo partido a PostgreSQL. La base de datos se genera y se adapta según se usa, sin que tú tengas que hacer nada. Además, las consultas se aceleran gracias al uso de Memcache.

Compatible

¿No usas PostgreSQL? No pasa nada. MySQL también está disponible para mayor compatibilidad y velocidad. Podrás instalar FacturaScripts en cualquier hosting.

Sencillo

La vida ya es bastante complicada como para estar dos horas para hacer una factura. Mira este pequeño tutorial para ver lo sencillo que es trabajar con FacturaScripts.

Extensible

FacturaScripts está desarrollado en PHP, uno de los lenguajes de programación más sencillos y extendidos. Y gracias al diseño de FacturaScripts, podrás añadir nuevas características de forma rápida y sin complicaciones. Mira este pequeño video-tutorial.
